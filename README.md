# Threads

Решил выполнить задание на Rust, который тоже позволяет управлять потоками.

Реализовал похожую логику SynchronizationContext.Send и SynchronizationContext.Post из C#

При вызове Post задача передается в очередь. Т.к. очередь в Rust работает по принципу много писателей, один читатель, то пришлось обернуть очередь в мьютекс, чтобы обеспечить потокобезопасность. Затем освободившийся поток берет задачу из очереди и выполняет ее.
